이 문서를 통해 플로깅 모집 사이트 쓰담의 API 사용법을 알 수 있습니다.

1. MemberController

1.1. 회원 등록

curl-request
$ curl 'http://localhost:8080/v1/members' -i -X POST \
    -H 'Content-Type: application/json;charset=UTF-8' \
    -H 'Accept: application/json' \
    -d '{
  "email" : "cjh@gmail.com",
  "password" : "code123!@",
  "confirmPassword" : "code123!@",
  "nickname" : "보노보노의습격단"
}'
